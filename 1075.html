I feel so dirty

When I started grad school I figured I’d never again have an opportunity to purchase affordable software so I purchased about everything that made sense. Mind you, I’ve been so busy I haven’t had time to use any of it.

However, tonight I did and so help me, it put Windows XP on my wonderful Apple PowerBook G4…. aaarrrgggghhh… I feel so dirty.

A friend and fellow Mac user brought over some CDs that for some reason could not be read on a Mac. He couldn’t read them and I couldn’t read them. So for the first time, I used Virtual PC and fired up Windows XP. Low and behold, I could read the discs.

I use Windows at work so that wasn’t a big deal, but it took me a good hour to figure out how Virtual PC worked so I find the files and burn him a new disk on the Mac side of things.

Maybe there’s a lesson in all this. Maybe it shows that sometimes you have to compromise yourself in order to attain the greater good. Even though I may hate it, I now know I can run Windows on my laptop if it means helping a friend. Is this what it’s like to co-sponsor legislation with a Republican? wink
